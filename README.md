## Framework
- streamlit == 1.37.1

## System Requirements
- Python >= 3.10 
- langchain == 0.2.12
- langchain-core==0.2.29
- langchain-community==0.2.11 

## Step to create local env
1. create venv 
2. install package `pip install -r requirements.txt`
3. copy .venv.sample to .env
4. Get API keys 
5. run app `streamlit run app.py`
